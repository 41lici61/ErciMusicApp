
# ErciMusic

## About
ErciMusic is an offline Android game where you guess songs from short audio snippets. Test your music knowledge across 6 levels with 30 songs!

## Features
- 6 progressive levels
- 30 songs from various genres
- Score & ranking system
- 3 lives per level
- Custom profiles with avatars
- Global leaderboard
- 100% offline - no internet needed

## Installation

### Download APK
1. Download from [Releases](https://github.com/41lici61/ErciMusicApp/releases)
2. Open APK and tap "Install"

## 🎮 How to Play

1. **Register** - Create account with nickname, password & avatar
2. **Login** - Enter your credentials
3. **Select Level** - Choose an unlocked level from the menu
4. **Play** - Tap ▶ to hear the song, select the correct answer
5. **Score Points** - +1 for correct, -1 life for wrong
6. **Level Up** - Accumulate enough points to unlock next level
